# LaporSekolah

EduLapor adalah platform pengaduan fasilitas pendidikan berbasis crowdsourcing yang dikembangkan menggunakan CodeIgniter 4. Sistem ini memungkinkan pengguna melaporkan kerusakan fasilitas pendidikan, admin memverifikasi laporan, serta petugas menindaklanjuti laporan secara transparan.

## Keterangan Proyek

Project ini dikembangkan sebagai bagian dari kegiatan magang di PT Vinix Seven Aurum. Sistem ini dibuat untuk memenuhi tugas pengembangan aplikasi berbasis web dengan tema pengaduan fasilitas pendidikan berbasis crowdsourcing.

## Fitur Utama

- Landing page publik
- Login dan register pengguna
- Dashboard berdasarkan role
- Role Admin, Petugas, dan Pelapor
- Pengelolaan kategori laporan
- Pengelolaan lokasi fasilitas
- Pembuatan laporan pengaduan
- Upload foto bukti laporan
- Verifikasi dan penolakan laporan oleh admin
- Penugasan laporan kepada petugas
- Update status laporan
- Komentar atau tanggapan petugas
- Laporan publik dan dukungan/vote
- Statistik dan lacak laporan

## Teknologi

- CodeIgniter 4
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Alur Sistem

Pelapor membuat laporan fasilitas yang bermasalah. Admin memverifikasi laporan dan menugaskan petugas. Petugas menangani laporan, memberikan komentar perkembangan, lalu menyelesaikan laporan. Pengguna dapat memantau status laporan secara transparan.
